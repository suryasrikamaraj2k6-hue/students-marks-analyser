# 📊 Student Marks Analyzer (Python)

## 📌 Description

This is a simple Python project that takes student marks as input and calculates the total, average, and grade.

## 🚀 Features

* Takes student name and marks as input
* Calculates total marks
* Calculates average
* Displays grade based on average

## 🛠️ Technologies Used

* Python

## ▶️ How to Run

1. Install Python on your system
2. Download or clone this repository
3. Open the file in any code editor
4. Run the program

## 📷 Sample Output

Enter student name: Ray  
Enter mark 1: 90  
Enter mark 2: 80  
Enter mark 3: 85  

   Result   
Name: Ray  
Total: 255  
Average: 85.0  
Grade: B
